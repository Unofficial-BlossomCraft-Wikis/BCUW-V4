# BCUW v4: Why?

The first time we switched from Astro Starlight, the build time was reaching a minute and a half. While that’s still better than v1 of the site, v3 wasn't fast enough. Although Docusaurus (the framework this site is built on) is still slow, it offers enough features to make the switch worthwhile.

# BCUW v4: What's Changed?

We have switched from Astro Starlight to Docusaurus for the site, but who cares about the backend? We just want to know about the frontend changes and new features! With this switch, not only do we now have a blog, but we also have a whole new layout and a new (better) search system— all designed to make your life easier.

# What's Coming in the Future?

Ah, the fun section! I have a lot of plans for the site, some of which are in alpha and some still in planning. So, they may not be released for a while, if ever, but here’s a list of my plans:

- A downloadable desktop app (planned)
- A community website designed for buying and selling items, and listing your player warps ([alpha](https://community.bcuw.xyz))
- A Minecraft mod that connects with the site (both the main docs and the community website) (planned)

Again, these are mostly ideas, so they may not be implemented.

# Short Term: What's Happening?

Within the next few months, I will be working to get the community website up and running and getting previous crates documented! I plan to have these done by the end of 2024!

That's all for now! Have a great day or night!
